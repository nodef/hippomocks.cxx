Hippomocks ![Build status](https://travis-ci.org/dascandy/hippomocks.svg?branch=master "Current build status on Travis CI")
==========

Single-header mocking framework, by [Peter Bindels](https://github.com/dascandy).


## Installation

Run:
```bash
$ npm i hippomocks.cxx
```

And then include `hippomocks.h` as follows:
```c
#include "node_modules/hippomocks.cxx/HippoMocks/hippomocks.h"
```
